## What I do
对指定文件添加代码注释, 具体需求如下:
1. 注释内容应当清晰、准确, 目标是让初学者能够立即理解代码运行的逻辑
2. 注释需要使用中文, 如果是专有名词, 请保持英文
3. 优先使用英文标点符号, 不要使用中文的 逗号/句号/括号!
4. 中文与英文之间请保留一个空格, 英文标点符号后也请保留一个空格, 例如: "这是 MoE 模型, 存在 2 个专家"
5. 已经存在的注释请保持不动
6. 不要修改任何代码逻辑, 即使你认为存在错误也不要修改
7. 除了注释之外, 不要添加任何额外的输出内容
8. 每次只标注一个函数 function 或一个类 class


## When to use me
当用户要求为代码文件添加注释的时候使用此技能
如果有任何不明白/不明确的情况, 请提出澄清性问题
